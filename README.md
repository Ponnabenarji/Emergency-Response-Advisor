## Project Description
The Emergency Response Advisor is a Python-based project that provides guidance during emergency situations such as fire, medical issues, or natural disasters.

## Features
- Identifies type of emergency
- Provides safety instructions
- Suggests immediate actions
- Simple and user-friendly

## Technologies Used
- Python
- Conditional Statements
- Functions
- Basic AI Logic

## How It Works
1. User enters the emergency type
2. System analyzes the input
3. Advisor gives step-by-step instructions
